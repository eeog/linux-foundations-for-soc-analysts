# Project Overview

This document provides a detailed explanation of the purpose and scope of the “Linux Foundations for SOC Analysts” project. The goal is to demonstrate essential Linux investigation skills used in Security Operations Centers, focusing on system navigation, file discovery, and command-line techniques.

These skills form the foundation of endpoint triage and incident response. Understanding how to move through the filesystem, inspect files, and search for indicators is critical for analysing suspicious activity on Linux systems.

This project is part of a broader portfolio designed to build toward full SOC-level investigations.
