# Skills Demonstrated

This project highlights several core competencies required for junior SOC analysts working with Linux systems.

## System Awareness
- Identifying the active user environment (`whoami`)
- Understanding directory context (`pwd`)

## Filesystem Navigation
- Moving through directories (`cd`)
- Listing directory contents (`ls`)
- Recognising hidden or unusual files

## File Inspection
- Reading file contents (`cat`)
- Reviewing configuration or text files for anomalies

## File Discovery
- Locating files by name (`find -name filename`)
- Searching by extension (`find -name "*.txt"`)

## Content Searching
- Searching inside files (`grep`)
- Recursively scanning directories for keywords (`grep -r`)

## Evidence Handling
- Redirecting output for documentation (`>`, `>>`)
- Saving findings for later review

## Workflow Efficiency
- Running commands in the background (`&`)
- Chaining commands for conditional execution (`&&`)

These skills support real-world SOC tasks such as triage, log analysis, and identifying suspicious activity.
